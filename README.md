# Guardweave Labs Website

Static landing page for Guardweave Labs Private Limited and its flagship product, PUFGuard.

## Files

- `index.html` — main landing page
- `styles.css` — responsive styling
- `script.js` — mobile navigation and demo contact-form behavior

## Before publishing

1. Replace the placeholder email address in `script.js` with the official Guardweave Labs contact email.
2. Add official company logo, CIN, address, privacy policy, and terms pages if required.
3. Connect the contact form to a backend or form service if you do not want to use `mailto:`.
4. Review all marketing claims against the final tested product data.

## Deployment options

This site can be deployed on GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static web hosting service.
